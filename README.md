# Student Productivity Analysis

## Objective
The goal of this project is to analyze how lifestyle factors such as study time, sleep, and phone usage impact student productivity.

## Dataset
Student behavior dataset including study hours, sleep hours, social media usage, stress level, and productivity score.

## Tools
- Python (pandas, seaborn, matplotlib)
- Tableau

## What I Did
- Performed exploratory data analysis to understand relationships between variables
- Built visualizations to identify key patterns and trends
- Analyzed correlations between study habits, stress, and productivity
- Designed dashboards to present insights in a clear and business-friendly way

## Key Findings
- Higher study hours are strongly associated with higher productivity
- Increased phone usage is negatively correlated with productivity
- Higher stress levels tend to reduce productivity
- Better sleep is linked to lower stress and improved performance

## Key Insight
This project shows that productivity is influenced by multiple lifestyle factors, and managing habits like study time and phone usage can significantly improve outcomes.
